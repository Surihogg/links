# Chrome Web Store 发布材料

## 商店名称
Links 快速收藏

## 简短描述（132 字符以内）
一键将当前网页收藏到 Links 桌面应用，配合本地链接管理工具使用。

## 详细描述

Links 快速收藏是 Links 桌面应用的官方浏览器扩展。

**使用前请确保已安装 Links 桌面应用。**

🔗 一键收藏
在任意网页点击扩展图标，即可将当前页面快速收藏到 Links。自动读取页面标题，无需手动输入。

🚀 零配置
安装即用，无需登录注册，无需填写令牌或端口。基于系统深度链接协议，安全可靠。

🛡️ 隐私安全
不收集任何用户数据，不访问浏览器历史，不与任何云端服务通信。所有数据存储在本地。

**如何使用：**
1. 安装 Links 桌面应用（https://github.com/Surihogg/links）
2. 安装本扩展
3. 在任意网页点击扩展图标 → Links 快速添加窗口自动弹出

**关于 Links 桌面应用：**
Links 是一款跨平台本地链接管理工具，数据完全存储在本地 SQLite 数据库，支持全文搜索、分组管理、标签系统、元数据自动抓取等功能。
