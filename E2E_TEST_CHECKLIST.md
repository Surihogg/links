## 附录：数据模型参考

### Link 字段
- `id` (i64), `url`, `title`, `description`, `notes`
- `favicon_url`, `og_image_url`
- `category_id` (Option), `is_favorite` (bool), `is_broken` (bool)
- `tags` (Vec\<String\>), `created_at`, `updated_at`

### Category 字段
- `id` (i64), `name`, `parent_id` (Option)
- `sort_order` (i32), `children` (Vec\<Category\>)
- `created_at`, `updated_at`

### Tag 字段
- `id` (i64), `name`, `updated_at`

### 配置项（config.json）
- `window-size` / `window-position` — 窗口尺寸和位置
- `global-shortcut` / `main-shortcut` — 快捷键
- `close-behavior` — 关闭行为（ask / tray / exit）
- `theme-mode` — 主题（light / dark / system）
- `http-server-token` — 书签小工具 token
- `spotlight-shortcut` — Spotlight 全局搜索快捷键
- `hide-shortcut` — 隐藏主窗口快捷键
- `auto-minimize` — 打开链接时自动最小化
- `auto-check-update` — 自动检查更新
